### Quick start
**Make sure you have Node version >= 8.0 and (NPM >= 5 or [Yarn](https://yarnpkg.com) )**
> Clone/Download the repo then edit `app.component.ts` inside [`/src/app/app.component.ts`](/src/app/app.component.ts)

```bash
# clone our repo
# --depth 1 removes all but one .git commit history
git clone git@github.com:cristian87dev/webee-ex-1.git

# change directory to our repo
cd client-src

# install the repo with npm
npm install

# start the server
npm start

# use Hot Module Replacement
npm run start:hmr
```
go to [http://0.0.0.0:3001](http://0.0.0.0:3001) or [http://localhost:3001](http://localhost:3001) in your browser
